添加评论

获取 APP ID 和 APP Key

首先需要注册一个 LeanCloud 账号，然后进入控制台，点击左下角创建应用。

随便起个名字，选择开发版，然后创建就可以了。创建完成之后，进入刚刚创建的应用，选择设置->应用Key，然后你就能找到 APP ID 和 APP Key 了。
